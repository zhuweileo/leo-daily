# Agent Skills

一个开放标准（agentskills.io），允许为 AI agent（如 [[Claude Code]]、Codex、Cursor、Gemini CLI 等 40+ 种）创建可复用的 skill。[[Second Brain]] 项目基于此标准，以 npm 包形式分发四个核心 skill。

通过 Agent Skills，用户只需 `npx skills add NicholasSpisak/second-brain` 即可在任何支持的 AI agent 中安装完整的知识库工具链。
