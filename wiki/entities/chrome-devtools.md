# Chrome DevTools

Google Chrome 内置的开发者工具集，提供网页调试、性能分析、网络请求检查、控制台日志查看等功能。在 [[Chrome DevTools MCP]] 项目中，其完整能力通过 [[MCP Protocol]] 暴露给 AI coding agent。

Chrome 144+ 版本支持 auto-connect 功能，允许 MCP Server 自动连接到用户已运行的 Chrome 实例，实现手动测试和 agent 驱动测试之间的状态共享。
